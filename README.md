Matrix Mech Co Static Website

This is a static HTML/CSS version of the site for easy deployment on Vercel.

Files:
- index.html
- styles.css
- README.md

How to deploy on GitHub + Vercel:
1. Upload these files to the root of your GitHub repo.
2. In Vercel, import the GitHub repo.
3. Framework preset can stay as "Other" or automatic.
4. No npm install is needed.
5. No build command is needed.

If Vercel asks for settings:
- Build Command: leave blank
- Output Directory: leave blank
- Install Command: leave blank
